# WHMCS Tickets On Hold / In Progress Widget

A simple **WHMCS Admin Dashboard widget** that displays all support tickets with status **"On Hold"** or **"In Progress"** â€” so administrators can always keep an eye on pending tasks.

---

## Features
- Displays tickets with status **On Hold** and **In Progress**
- Shows ticket ID, subject, department, last reply, and assigned admin
- Automatically refreshes on page load
- Clean and responsive design for the admin area
- Lightweight and fully integrated into WHMCS widgets system

---

## Requirements
- WHMCS v8.12 or higher
- Admin access to WHMCS installation
- PHP 8.1+ or newer
